# Dub Siren PCB

PCB design for a dub siren.

## Files
- `Altium_Project_Dub_Siren.zip` — full Altium Designer project
- `Dub_Siren_Fabrication_Files.zip` — Gerbers + NC drill
- Parts list and build notes included
